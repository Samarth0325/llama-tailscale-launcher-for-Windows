# 🚀 LLaMA + Tailscale Auto Launcher

---

## ✨ Features

* 🔹 One-click setup (auto install dependencies)
* 🔹 Auto connect to Tailscale
* 🔹 Multi-model support (.gguf)
* 🔹 Runs in background (no terminal needed)
* 🔹 Auto port management (8080)
* 🔹 Works with custom control server (Headscale)

---

## 📦 Requirements

* Windows 10/11
* Admin access
* Internet connection
